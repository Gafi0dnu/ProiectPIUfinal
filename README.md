# ProiectPIUfinal
Tema 5 consta in realizarea unui sistem de gestionare pentru o companie de transporturi. Sistemul ar trebui sa permita: adaugarea, editarea si stergerea soferilor si masinilor introduse, alocarea soferilor o tura (tura 1, tura 2 etc), vizualizarea numarului de km parcursi de fiecare masina sau sofer.
